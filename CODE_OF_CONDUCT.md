# Código de Conducta

Todos los participantes, colaboradores y mantenedores de SARA se comprometen a respetar el siguiente código de conducta.

Puntos clave:
- Sé respetuoso y considerado con los demás.
- No toleramos acoso, lenguaje discriminatorio ni ataques personales.
- Acepta la retroalimentación y mantén conversaciones constructivas.

Si experimentas o presencias una conducta inadecuada, contacta a los mantenedores del proyecto.
